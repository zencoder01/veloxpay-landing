## 🚀 Common Tasks

### See the Landing Page
```bash
npm run dev
# http://localhost:3000
```

### Change Company Name
Search for "VeloxPay" and replace with your company name

### Change Colors
- Primary green: `green-400` → `your-color-400`
- Secondary cyan: `cyan-400` → `your-color-400`
- Background: `#0f1629` → your color

### Update Copy
- Hero headline: `app/components/hero.tsx`
- Features: `app/components/features.tsx`
- Footer: `app/components/footer.tsx`

### Deploy to Vercel
1. Push to GitHub
2. Go to vercel.com
3. Connect repository
4. Done! (Automatic deployments)
